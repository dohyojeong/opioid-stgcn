# Spatio-Temporal Graph Convolutional Network (ST-GCN) 

This repository contains code for a spatio-temporal graph convolutional network (ST-GCN)
used for monthly grid-level prediction of opioid-involved mortality.

## Contents
- preprocessing/: grid construction, rook adjacency, monthly aggregation
- models/: ST-GCN, spatial GCN, LSTM baselines
- training/: training scripts
- evaluation/: metrics and evaluation utilities

Ongoing development for abstract submission; code will be updated.
